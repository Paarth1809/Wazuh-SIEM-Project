# 🛡️ Wazuh SIEM Project

## 📌 Overview
This project demonstrates a Security Information and Event Management (SIEM) setup using Wazuh on an AWS Ubuntu server. Agents are configured on Windows 10 and Kali Linux to monitor system activity and detect security threats.

---

## 🏗️ Architecture
- Wazuh Server (AWS): 52.66.238.137  
- Windows Agent: 192.168.153.128  
- Kali Linux Agent: 192.168.253.130  

---

## ⚙️ Features
- File Integrity Monitoring (FIM)  
- Windows Registry Monitoring  
- SSH Brute Force Detection  
- Vulnerability Detection  
- Centralized Log Monitoring  

---

## 🚀 Tools Used
- Wazuh  
- AWS EC2 (Ubuntu)  
- Windows 10  
- Kali Linux  

---

## 📊 Outcome
Successfully implemented centralized monitoring and detected file changes, registry modifications, SSH attacks, and vulnerabilities in real-time.
